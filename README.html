How to use this pack with your students

  1. Week 0 — Orientation. Hand out the master study-calendar.md. Walk students through how the BISE Abbottabad paper is structured (sections, marks, time, attempt rules) using each study-guide.md's "paper pattern" section.
  2. Weeks 1–8 — Topical mastery. Follow the calendar. For each subject, work high-weightage chapters first (marked High in the weightage tables). Use the "Top guess topics" section to prioritize what must be memorized vs. understood.
  3. Week 9 — Mock exam. Print each model-paper-1.md and have students attempt under timed conditions (3 hours per paper). Mark using answer-key.md. The marking scheme follows the same rubric the board uses.
  4. Week 10 — Targeted revision. Use the mistake patterns surfaced during mock marking to revisit weak chapters.

Note: The exact split per session is published by BISE Abbottabad on the official sample-papers page. If 2026 changes the count of MCQs or SRQs, only the cover sheet of each model paper needs an edit — the questions themselves remain valid.

How the high/medium/low weightage labels were assigned

The trend labels in each study-guide.md come from cross-referencing:

  • KPTBB Table of Specifications (ToS) for each subject — the official mark allocation per chapter.
  • Repeat patterns from BISE Abbottabad past papers 2019–2025 as listed on the source pages you provided.
  • Common questions in the official model paper 2024 released by BISE Abbottabad.

Where the ToS and past-paper trend disagree, trend wins — students prepare for what is actually asked.
