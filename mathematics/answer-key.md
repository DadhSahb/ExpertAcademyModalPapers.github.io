## Section B — Short Response (4 marks each, attempt any 10)

**B1.** Multiplicative inverse of `3 − 4i`:
`1/(3 − 4i) × (3 + 4i)/(3 + 4i) = (3 + 4i)/(9 + 16) = 3/25 + (4/25)i`. *(2 marks rationalisation, 2 marks final form.)*

**B2.** `(A')' = A`. Take `A = [[1,2],[3,4]]`, then `A' = [[1,3],[2,4]]`, `(A')' = [[1,2],[3,4]] = A`. *(1 mark statement, 3 marks verification.)*

**B3.** Let `y = x²`. Then `y² − 5y + 4 = 0 → (y−1)(y−4) = 0 → y = 1 or 4 → x = ±1, ±2`. *(2 marks substitution, 2 marks roots.)*

**B4.** `(7x + 3)/[(x+1)(x−2)] = A/(x+1) + B/(x−2)`. Cover-up: `A = (7(−1)+3)/(−1−2) = −4/−3 = 4/3`; `B = (7(2)+3)/(2+1) = 17/3`. Answer: `4/[3(x+1)] + 17/[3(x−2)]`. *(2 marks setup, 2 marks values.)*

**B5.** A.P.: `aₙ = a + (n−1)d`. From `a + 3d = 13`, `a + 8d = 28` → `5d = 15 → d = 3, a = 4`. So `a₁₂ = 4 + 11(3) = 37`. *(2 marks system, 2 marks answer.)*

**B6.** G.P.: `a = 2`, `r = 3`, `aₙ = 4374`. `2 · 3ⁿ⁻¹ = 4374 → 3ⁿ⁻¹ = 2187 = 3⁷ → n = 8`. `S₈ = 2(3⁸ − 1)/(3 − 1) = (6561 − 1) = 6560`. *(2 marks find n, 2 marks sum.)*

**B7.** EQUATION has 8 distinct letters → `8! = 40320` arrangements. *(1 mark count, 3 marks compute.)*

**B8.** P(king) = 4/52, P(queen) = 4/52, mutually exclusive → P = 8/52 = **2/13**. *(2 marks events, 2 marks answer.)*

**B9.** *n = 1:* LHS = 1, RHS = 1·2/2 = 1 ✓. *n = 2:* LHS = 1 + 2 = 3, RHS = 2·3/2 = 3 ✓. *(2 marks each.)*

**B10.** `Tᵣ₊₁ = C(10, r) · x¹⁰⁻ʳ · (1/x)ʳ = C(10, r) x¹⁰⁻²ʳ`. For independence, `10 − 2r = 0 → r = 5`. Term = `C(10, 5) = 252`. *(2 marks setup, 2 marks compute.)*

**B11.** `225° × π/180 = 5π/4 rad`. *(2 marks conversion factor, 2 marks answer.)*

**B12.** LHS = `(sin A cos B + cos A sin B)(sin A cos B − cos A sin B) = sin²A cos²B − cos²A sin²B`. Substitute `cos²B = 1 − sin²B` and `cos²A = 1 − sin²A` and simplify → `sin²A − sin²B`. *(2 marks expand, 2 marks simplify.)*

**B13.** Use law of cosines: `c² = a² + b² − 2ab cos C = 49 + 25 − 70(0.5) = 39 → c ≈ 6.24 cm`. Then `sin A / 7 = sin 60° / 6.24 → A ≈ 76.0°`. So `B = 180 − 60 − 76 = 44°`. *(3 marks for c, 1 mark each angle.)*

**B14.** Period = 2π, amplitude = 2. Sketch a sine wave touching y = ±2 with one full cycle in 0 to 2π. *(2 marks correct shape, 2 marks labels.)*

**B15.** `sin⁻¹(−1/2) = −π/6` (principal value). *(4 marks.)*

**B16.** `cos² θ = 1/2 → cos θ = ±1/√2 → θ = π/4, 3π/4, 5π/4, 7π/4`. *(2 marks setup, 2 marks all four values.)*

**B17.** `tan(45+A) tan(45−A) = [(1+tan A)/(1−tan A)] · [(1−tan A)/(1+tan A)] = 1`. *(4 marks, full simplification.)*

**B18.** Group: a non-empty set with binary operation satisfying closure, associativity, identity, and inverse. For `(Z, +)`: closure (sum of integers is integer ✓), associative ✓, identity = 0 ✓, inverse of `a` is `−a` ✓. Hence `(Z, +)` is a group. *(2 marks definition, 2 marks verification.)*

**B19.** `nC(r) + nC(r−1) = n!/[r!(n−r)!] + n!/[(r−1)!(n−r+1)!]`. Take LCD `r!(n−r+1)!` and combine → `(n+1)!/[r!(n+1−r)!] = (n+1)C(r)`. *(2 marks LCD, 2 marks simplification.)*

**B20.** `det(A) = 8 − 3 = 5`. `A⁻¹ = (1/5) · [[4, −1], [−3, 2]] = [[4/5, −1/5], [−3/5, 2/5]]`. *(2 marks det, 2 marks adjoint & answer.)*

**B21.** `A ∪ B = {1,2,3,4,6}`, `(A ∪ B)' = {5}`. `A' = {4,5,6}`, `B' = {1,3,5}`, `A' ∩ B' = {5}`. Equal ✓. *(2 marks LHS, 2 marks RHS.)*

**Q23 (main).**
